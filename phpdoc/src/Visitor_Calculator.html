Class Visitor_Calculator

Class Overview
Class Overview | Property Summary | Properties Detail | Method Summary | Methods Detail

Finds the maximum number of visitors and the time ranges

Uses the Filereader to read the times from a file and saves the times in minutes in the _visitors array. Can then be used to find the maximum number of visitors and the time ranges.

	findMax  [line 107]





	

    

    
 
		int findMax(
)
    

    



	
Finds the maximum number of simultaneous visitors in the museum



Goes through the _visitor array and marks the times when the visitor  has been in the museum by increasing the corresponding numbers in the  _minuteArray. Saves the number of visitors to the _max variable whenever  the _max variable is smaller than the current number in the _minuteArray  array.
